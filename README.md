Virtual Keyboard

A simple and interactive **Virtual Keyboard** built using Python and Tkinter.  
This project simulates a real keyboard on the screen and allows users to type using mouse clicks.

 🚀 Features

- On-screen virtual keyboard layout  
- Clickable keys for input  
- Text appears in a display box  
- Supports alphabets, numbers, and special keys  
- Easy to customize and extend  
- Lightweight and beginner-friendly project

 🛠️ Technologies Used

- **Python**
- **Tkinter** (GUI Library)
